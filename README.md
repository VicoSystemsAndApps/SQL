# Vico Homes – Data & Analytics Code Repository

## Overview
This repository is used by **Vico Homes** to store, manage, and version control code and assets related to ongoing data, analytics, and reporting projects.

It provides a single, governed location for development work across:
- SQL Server and Azure SQL
- Microsoft Fabric
- Azure (data and analytics services)
- Power BI

The repository supports collaboration, consistency, and reuse across project teams.
